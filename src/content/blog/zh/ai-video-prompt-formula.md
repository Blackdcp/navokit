想让 AI 视频提示词更稳定，最简单的方法不是写得更长，而是从“描述一整支视频”改成“描述一个清晰镜头”。

文生视频系统更适合处理聚焦的视觉目标。如果你在一个短视频生成任务里要求完整广告、多场景、对白、字幕、Logo 结尾和复杂剪辑，结果很容易混乱。如果你只描述一个主体、一个动作、一个场景和一个镜头方向，生成结果通常更可用。

## 六段式提示词公式

使用这个结构：

```text
主体 + 动作 + 场景 + 镜头运动 + 光线 + 风格
```

不是每次都必须包含全部部分，但这个公式能帮你避免模糊输入。

示例：

```text
一只红狐走过积雪森林，低机位跟拍，雪花轻落，阴天自然光，写实电影风格。
```

它为什么有效：

- **主体**：一只红狐
- **动作**：走过
- **场景**：积雪森林
- **镜头运动**：低机位跟拍
- **光线**：阴天自然光
- **风格**：写实电影风格

每个部分都给模型一个可见目标。

## 1. 主体

主体是画面里的主要对象，可以是人物、动物、产品、物体、空间或抽象视觉。

好的写法：

```text
一个陶瓷咖啡杯
```

太模糊：

```text
一些很舒服的东西
```

如果主体很重要，就直接写清楚。不要一次堆太多主体，除非它们之间关系很简单。

## 2. 动作

动作告诉模型画面中应该发生什么变化。

示例：

- 走过雪地
- 咖啡热气升起
- 薄雾围绕香水瓶流动
- 彩色便利贴铺满白板
- 镜头缓慢推向产品

一个动作通常就够了。如果提示词里出现“然后”“接着”“切到”，它可能已经在尝试做太多事情。

## 3. 场景

场景给主体提供上下文，也会影响色彩、材质、背景和情绪。

示例：

- 黑色石材表面
- 安静咖啡馆里
- 干净白色桌面
- 日出时的雾气森林
- 深色玻璃背景

产品镜头里，场景常常是一个表面。电影感镜头里，场景通常是地点和时间。

## 4. 镜头运动

镜头运动应该简单。清楚的镜头语言能给结果方向，但不会压垮模型。

常用写法：

- 缓慢推进
- 固定机位
- 低机位跟拍
- 俯拍
- 微距特写
- 广角静态镜头
- 缓慢环绕
- 轻微横移

不要把太多镜头词堆在一起。“缓慢推进”通常比“动态电影感镜头、快速推拉、手持摇晃”更稳定。

## 5. 光线

光线决定画面情绪。相比“高级”“好看”这类抽象词，明确光线往往更有用。

示例：

- 柔和晨光
- 戏剧化轮廓光
- 冷蓝色棚拍灯光
- 温暖桌面氛围光
- 金色日落逆光
- 阴天自然光

如果生成结果显得平，可以加入光线方向、颜色或一天中的时间。

## 6. 风格

风格告诉模型你想要哪种视觉语言。风格要具体，但不要过载。

好的写法：

- 写实电影风格
- 极简产品广告风格
- 安静生活方式镜头
- 奢华产品摄影风格
- 适合落地页的轻微未来感背景

较弱写法：

- 史诗级爆款大片
- 超级高级好看
- 做得漂亮一点

模型需要可见指令。风格有帮助，但不能替代具体细节。

## 不同场景的提示词示例

产品镜头：

```text
黑色岩石上的香水瓶，薄雾缓慢流动，镜头缓慢推进，戏剧化轮廓光，奢华产品广告风格。
```

创作者工作台：

```text
夜晚的舒适桌面，笔记本屏幕发光，咖啡热气升起，镜头缓慢推进，温暖氛围光，安静创作者工作区。
```

网站首屏背景：

```text
抽象蓝色光波在深色玻璃表面缓慢流动，动作平滑，极简未来感风格，适合作为落地页背景。
```

社交短视频钩子：

```text
彩色便利贴铺满白板，俯拍镜头，明亮办公室光线，有活力的效率主题视觉。
```

电影感自然场景：

```text
一个骑行者穿过日出时的雾气桥面，广角镜头，柔和蓝色光线，缓慢跟拍，写实电影感。
```

如果生成失败或等待太久，先简化提示词，再考虑更长时长。

## 常见错误

最常见的错误，是在一个提示词里生成多个场景。短视频生成不是完整分镜脚本。

第二个错误，是要求精确可读文字。AI 视频工具不一定能稳定生成准确字体和排版。如果需要文字，最好后期再加。

第三个错误，是形容词太多。“高级电影感惊艳大片”不如“黑色石材表面、薄雾、戏剧化轮廓光”有用。

最后，不要让模型替你决定创意策略。先明确这段视频要完成什么任务，再写镜头。

## 检查生成结果

生成后，像剪辑师一样检查：

- 主体是否清楚？
- 动作是否稳定？
- 镜头运动是否符合提示词？
- 光线是否接近目标情绪？
- 画面比例是否适合平台？
- 是否出现奇怪物体或变形？
- 不解释时，这段视频是否仍然能看懂？

如果结果接近目标，只改一个小细节。如果偏差很大，就简化提示词，回到一个可见想法。
