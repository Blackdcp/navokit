## 为什么要把 Markdown 转成图片？

Markdown 适合写作和整理信息，但很多社交平台、聊天工具和视觉内容场景无法保留 Markdown 格式。转成 PNG 后，内容更容易分享、归档和复用。

适合转换的内容包括：

- AI 回答
- 技术笔记
- 版本摘要
- 金句卡片
- Newsletter 片段

## 第一步：整理 Markdown

先让内容结构清楚：

```markdown
# 项目笔记

**目标：** 让工作流更容易理解。

- 输入清楚
- 结果可见
- 输出可复用
```

短段落通常比长篇文字更适合导出成图片。

## 第二步：粘贴到 NavoKit

打开 [Markdown 转图片](/zh/tools/markdown-to-image)，把 Markdown 粘贴到编辑器。预览会实时更新，你可以检查标题、列表、引用、链接和代码块。

渲染在浏览器中完成，粘贴的 Markdown 不会上传到 NavoKit 服务端。

## 第三步：检查图片卡片

导出前检查：

- 标题是否清楚？
- 列表是否容易扫读？
- 代码块是否可读？
- 图片是否过长？

如果内容太密，可以拆成两张较短的卡片。

## 第四步：导出 PNG

点击导出即可下载 PNG。免费版本会包含克制的 NavoKit 品牌标识，用于支持工具持续运营。
