AI 视频生成器已经强得离谱了。但大多数人输入的是"一个漂亮的日落"，然后纳闷为什么生成的画面那么平庸。

问题几乎从来不在模型，而在提示词。

我们测试了数百条提示词后发现，一段平庸的 AI 视频和一段电影级画面之间的差距，归结为四个要素。我们把它叫做 **SMSC 框架**：Subject（主体）、Motion（动作）、Style（风格）、Camera（镜头）。

## SMSC 框架

每一条强有力的 AI 视频提示词都包含四个层次。少了任何一层，模型就会用自己的默认值来填补——而默认值通常很无聊。

| 层次 | 控制什么 | 弱示例 | 强示例 |
|------|---------|--------|--------|
| **主体 Subject** | 画面中有谁/有什么 | "一个人" | "一位白发科学家，身处堆满仪器的实验室" |
| **动作 Motion** | 正在发生什么，什么在运动 | （未指定） | "缓缓将一管发光的蓝色液体倒入烧杯" |
| **风格 Style** | 视觉美学与情绪 | "电影感" | "韦斯·安德森配色，柔和粉彩光线，对称构图" |
| **镜头 Camera** | 景别与摄影机运动 | （未指定） | "35mm 镜头缓慢推入，浅景深" |

差距是巨大的。左边生成的是一张会动的 AI 图库照片。右边生成的是一段看起来有意图的影像。

## 可直接复制的提示词模板

以下是 7 个开箱即用的提示词模板，每个对应一种常见的使用场景。把它们复制到任何文生视频工具中（比如我们的[免费 AI 视频生成器](/zh/tools/free-ai-video-generator)），替换方括号里的变量即可。

### 模板 1：产品展示镜头
```
一个 [产品名] 放置在 [材质] 桌面上。
柔和的顶光打出干净的阴影。
摄影机缓缓绕产品旋转 180 度。
影棚灯光，白色背景，商业摄影风格。
```

### 模板 2：科技产品/SaaS 片头
```
抽象 3D 可视化展示 [概念，如"数据在神经网络中流动"]。
发光粒子沿着半透明路径运动。
摄影机后拉，揭示完整结构。
深色背景，电光蓝与紫色点缀，未来感 UI 美学。
```

### 模板 3：社交媒体开场（竖版）
```
[人物/角色] 的面部特写，直视镜头，表情 [情绪]。
缓缓挑起一侧眉毛。
浅景深，温暖的黄金时段光线。
竖版 9:16 格式，抖音/TikTok 原生质感。
```

### 模板 4：电影级风光（YouTube 封面 / B-roll 素材）
```
[地点] 的航拍无人机画面。
摄影机在日出时分掠过 [地形特征] 向前滑行。
体积雾，丁达尔光线，超宽银幕 2.39:1 画幅。
ARRI Alexa 拍摄质感，自然调色。
```

### 模板 5：氛围短片（叙事向）
```
一个孤独的 [角色] 在 [时间] 走在 [环境] 中。
细雨轻落，霓虹灯牌倒映在水洼里。
摄影机从背后远距离跟随，手持微晃。
《银翼杀手 2049》色调，变形宽银幕镜头光晕。
```

### 模板 6：教育/图解动画
```
干净的 2D 动画展示 [流程，如"Transformer 模型如何处理 Token"]。
元素逐个出现，带有平滑缓动效果。
白色背景，扁平设计，主色调 [颜色名或色号]。
俯视正交视角，摄影机不动。
```

### 模板 7：整活 / 病毒式传播
```
一只 [动物] 穿着 [反差服装] 坐在企业办公室的工位上。
它转头面向摄影机，面无表情地凝视。
日光灯管照明，安防摄像头视角，轻微颗粒噪点。
固定广角镜头，摄影机不动。
```

## 为什么结构化提示词的效果差这么多

我们在[免费 AI 视频生成器](/zh/tools/free-ai-video-generator)中测试了每个模板，并将输出与描述同一场景的单行提示词进行对比。结构化提示词始终表现出：

- **更连贯的运动** — 指定"什么在动"能避免"所有东西随机扭曲"的问题。
- **更好的光线** — 给出光线风格（"黄金时段"、"影棚顶光"）让模型有了强锚点。
- **有意图的构图** — "缓慢推入"或"固定广角"这类镜头指令消除了 AI 视频最常见的毛病："对着空气缓慢放大"。

## 镜头语言速查表

如果你没有影视背景，以下是 AI 视频模型响应效果最好的镜头术语速查：

| 术语 | 效果 | 最适合 |
|------|------|--------|
| 推入 Push-in | 摄影机向主体靠近 | 制造紧张感、揭示细节 |
| 后拉 Pull-back | 摄影机远离主体 | 建立场景全貌 |
| 环绕 Orbit | 摄影机围绕主体旋转 | 产品展示、英雄时刻 |
| 跟拍 Tracking | 摄影机与主体并行移动 | 角色行走、运动场景 |
| 固定广角 Static wide | 摄影机不动，广角镜头 | 喜剧、纪录片、整活 |
| 手持 Handheld | 轻微晃动，亲密感 | 叙事、真实感 |
| 航拍 Drone / aerial | 高空俯瞰，向前推进 | 风光、转场 |
| 特写 Close-up | 面部或细节紧凑取景 | 情绪、质感 |
