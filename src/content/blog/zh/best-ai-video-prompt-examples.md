好的 AI 视频提示词通常不长，但很具体。它应该描述一个短镜头，而不是一整支广告或一段完整故事。你需要告诉模型：画面主体是谁、它在做什么、场景在哪里、镜头怎么动、光线和风格是什么。

如果提示词太宽泛，结果会随机；如果一次写了多个场景，模型可能压缩、忽略，或者把不同画面混在一起。短视频提示词最重要的不是“写得多”，而是“写得清楚”。

## 提示词公式

可以使用这个结构：

```text
主体 + 动作 + 场景 + 镜头运动 + 光线 + 风格
```

例如：

```text
一只红狐走过积雪森林，雪花轻落，低机位跟拍，阴天自然光，写实电影风格。
```

这个提示词的好处是信息清楚：主体是红狐，动作是行走，场景是雪地森林，镜头是低机位跟拍，氛围是自然、写实、电影感。

## 产品镜头提示词

产品镜头要特别明确物体、表面、光线和镜头运动。一个产品放在一个清楚的环境里，通常比复杂陈列更稳定。

```text
一瓶香水放在黑色石材上，薄雾缓慢飘过，镜头慢慢推进，强烈轮廓光，高级商业广告风格。
```

```text
一个陶瓷咖啡杯放在木质书桌上，窗外晨光照入，镜头缓慢推进，安静生活方式风格。
```

```text
一双跑鞋放在湿润柏油路面上，表面有水珠，低角度跟拍，冷蓝色光线，运动品牌广告风格。
```

```text
无线耳机盒在干净白色桌面上打开，柔和棚拍光，微距慢镜头，极简科技产品风格。
```

改写时可以先替换产品，再替换表面和光线。镜头运动保持简单：推进、平移、环绕、跟拍或固定镜头即可。

## 社交短片提示词

社交短片更看重“第一眼是否能理解”。一个强视觉往往比复杂叙事更有效。

```text
夜晚的创作者书桌，笔记本屏幕发光，咖啡热气上升，镜头缓慢推进，温暖环境光，安静工作氛围。
```

```text
一名创作者在干净桌面上整理小型相机包，俯拍视角，动作快速但平滑，自然日光，生活方式风格。
```

```text
彩色便利贴在白板上逐渐展开，俯拍快速运动，明亮办公室光线，高效工作氛围。
```

```text
手机放在桌面上弹出日程提醒，近景特写，柔和晨光，干净极简办公桌，写实风格。
```

这类视频可以作为短帖背景、开头画面或说明型内容的视觉引子。生成画面后，可以用 [AI 社交媒体文案](/zh/tools/ai-social-booster) 起草配套文案，再按自己的语气修改。

## 电影感场景提示词

想要电影感，不要只写“cinematic”。更好的方式是描述光线、景别、情绪和镜头运动。

```text
一名骑行者穿过清晨有雾的桥面，远景，柔和蓝色光线，缓慢跟拍，写实电影风格。
```

```text
夜晚霓虹街道上雨水落下，湿润路面反射灯光，镜头缓慢移动，情绪化电影画面。
```

```text
一艘小船漂过清晨安静湖面，远景固定镜头，淡橙色天空，薄雾，诗意电影风格。
```

```text
一个人站在沙漠公路边，风吹动外套，广角远景，金色黄昏光，镜头缓慢拉近。
```

如果结果过于夸张，减少“史诗、震撼、大片”等抽象词，多写具体可见的画面细节。

## 网站和落地页背景提示词

网站背景视频不应该太抢眼。它通常需要干净、可循环、不会干扰文字阅读。

```text
深色玻璃表面上流动的蓝色光波，缓慢平滑运动，极简未来感，适合作为落地页背景。
```

```text
几张白色纸卡在桌面上整齐滑入，俯拍视角，柔和阴影，干净效率工具风格。
```

```text
细小发光粒子在深色工作空间中缓慢移动，微弱蓝光，极简科技背景。
```

```text
空白笔记本在桌面上打开，旁边放着一支笔，俯拍慢镜头，柔和自然光，安静编辑风格。
```

如果视频要放在标题后面，生成后一定要搭配文字预览。画面再好，如果让标题难以阅读，也不适合作为网页素材。

## 不建议的提示词写法

不要一次要求太多内容：

```text
做一支 30 秒广告，包括产品介绍、三个生活方式场景、用户反馈、文字字幕和最后 Logo。
```

不要依赖精确文字：

```text
画面里出现一个写着“Launch Sale 50% Off”的牌子，文字必须完美。
```

不要太抽象：

```text
做一个关于效率工具的爆款 AI 视频。
```

可以改成：

```text
一张凌乱书桌逐渐变得整齐，俯拍视角，动作平滑，明亮晨光，高效工作的视觉隐喻。
```
